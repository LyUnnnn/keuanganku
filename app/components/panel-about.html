Panduan Penggunaan

Cara setup dan menggunakan KeuanganKu

Alur Penggunaan

① Setup sekali — Masukkan URL Apps Script di halaman Pengaturan

② Input transaksi — Pilih jenis, isi form, klik Kirim

③ Hutang & Piutang — Gunakan panel khusus untuk sheet Hutang dan Piutang

④ Data langsung masuk ke Google Sheets Anda secara real-time

⑤ Lihat Saldo di panel Saldo & Ringkasan untuk monitor keuangan

⑥ Install sebagai PWA untuk akses lebih cepat dari home screen

Kolom di Google Sheets

A — Timestamp (otomatis)

B — Tanggal Input

C — Deskripsi

D — Kategori

E — Jenis (Masuk/Keluar/Netral)

F — Nominal

G — Sumber Uang

H — Kelompok Kategori

Kolom Hutang & Piutang

A — Timestamp / No (opsional)

B — Tanggal Input

C — Jatuh Tempo

D — Deskripsi

E — Pemberi Utang / Piutang

F — Nominal

G — Status

H — Pengingat

I — Jenis Utang

Panel Saldo — Cara Kerja

• Panel Saldo membaca langsung dari Google Sheets via Apps Script

• Pastikan fungsi doGet mendukung parameter ?action=saldo

• Gunakan template Apps Script yang disiapkan di dokumentasi proyek

• Data diperbarui setiap kali klik tombol Refresh

Tips Kecepatan

• Klik jenis transaksi (Masuk/Keluar/Netral) terlebih dahulu

• Kategori akan otomatis mengisi Kelompok Kategori

• Data tersimpan lokal jika koneksi terputus

• Export CSV tersedia lewat tombol 📤 di atas
