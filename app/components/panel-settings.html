Google Sheets — Apps Script Disimpan di server

Pengaturan disimpan di backend server (Node.js). Semua perangkat yang login akan otomatis mendapat konfigurasi yang sama. Hanya Admin yang dapat mengubah pengaturan.

Nama Sheet

Keamanan PIN

Dua mode akses: Admin (edit & lihat semua data) dan User (hanya input transaksi, saldo tersembunyi).

PIN Admin
Akses penuh — lihat saldo, edit, hapus data
PIN User
Input transaksi saja — saldo tidak terlihat
Kode Apps Script

1. Buka Google Spreadsheet Anda

2. Klik Extensions → Apps Script

3. Paste kode di bawah, lalu Save

4. Klik Deploy → New deployment → Web App

5. Set Execute as: Me dan Who has access: Anyone

6. Copy URL deployment ke pengaturan di atas
